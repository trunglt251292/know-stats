# Know Stat

[knowstats-reporter](https://github.com/trunglt251292/know-stats) is a backend service (Socket IO)

# Dependencies
- git
- npm
- nodejs

# Install
- git clone https://github.com/trunglt251292/know-stats.git
- npm install
- Setup file config.
- npm start:production
